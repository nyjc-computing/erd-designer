# Changelog

## [1.1.0] - 2026-02-22

### Added
- **File System Access API**: Support for direct file overwriting and "Save" permissions.
- **Collapsible Sidebar**: Toggleable UI to maximise canvas workspace.
- **Hotkeys**: Added `Ctrl+S` (Save), `Ctrl+D` (Duplicate), and `Delete` functionality.
- **View-Aware Placement**: New entities now spawn in the centre of the current viewport.

### Fixed
- **Session Undo**: Reverting a "Clear All" action now correctly restores the linked file handle.
- **Initial Visibility**: Projects now automatically "Zoom to Fit" upon being opened.

### Changed
- Refactored documentation and UI strings
